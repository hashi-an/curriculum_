ログ(意味、確認の仕方)

  • ログとは、コンピュータの利用履歴を記録したデータのこと。
    「いつ」「誰が」「何をしたか」を記録し、そのデータを蓄積することができ、収集したログを分析すれば、目的に合わせてデータを活用することも可能。
  • windowsでの確認の仕方：コントロールパネル内のイベントビューアからログを得る
  • PHPでの確認の仕方：php.iniによる設定や、error_log関数により、ログ内容を得たり、ファイルに出力して確認する。
  • git/gitHubでの確認の仕方：ブラウザ画面上の情報もしくは、git logコマンドで得る。

クラウドサーバー、レンタルサーバー、VPSの違い

ネットワーク経由で使う形態になっているサーバのこと

  • 【クラウドサーバー】1台のサーバーに加速化した専用サーバーを個別に立てる（大規模向け）ネットワーク経由で使える仮想サーバ、SaaS/PaaS/IaaSのサービスが提供される。
    Amazon Web Services（AWS）/Microsoft Azure/IBM Cloud/Google Cloud Platformなど
  • 【レンタルサーバー】1台のサーバーを複数サイトで共用する、サーバーをレンタル（貸し出す）するサービス（小規模・個人向け）
    エックスサーバー/ロリポップ/mixhostなど
  • 【VPS（仮想専用サーバー）】１台の物理サーバー上に複数の仮想サーバー領域を設定して、複数のユーザーでそれぞれのサーバー領域を使用するサーバー（中規模向け）。1台の物理サーバーを複数人で共有する点では、一般的なレンタルサーバー（共用サーバー）に似ています。しかし、VPSの場合はそれぞれの仮想サーバー領域が独立しているため、お互いの影響を受けにくく、専用サーバーのように柔軟にカスタマイズできるのが大きな特長。

PHPのFWの種類と特徴

PHPフレームワークとは、PHPを使用してWebアプリケーション開発する際の土台として機能するソフトウェア（クラスやライブラリの集合）のこと。 つまり、共通するコードに機能性を加えて形成された骨組み、枠組み。

  • Laravel：コードが美しく書け近年人気、日本語ドキュメントもありGitHubでも好評価。
  • CakePHP：Ruby on Railsの影響を受けており、プログラム自動生成機能があるため、素早く開発できる。
  • FuelPHP：シンプルで学習の難易度が低く記述ルールが少ないため自由度が高く統一性がとりにくい、大人数が関わる開発には不向き。
  • Symfony：デバックツールや機能テストなどが標準装備されている多機能さが強み、大規模な開発を得意としている。

CakePHPの2系・3系の違い

以下の改善点により、CakePHP 3は、CakePHP 2に比べて柔軟で強力なフレームワークとなった。

  • 名前空間の使用：PHPの名前空間を使用してコードを整理することができます。これにより、クラス名の重複を回避できるようになった。
  • コンフィグレーションの改善：マッチングが改善された。
  • ORMの改良：ORM（Object Relational Mapping）が大幅に改良され、モデルとデータベースの関係がより柔軟になった。ORMの新機能には、ビヘイビア、エンティティ、およびベルクラスが含まれる。
  • アダプターシステムの改善：Twigというフォントエンジンを使用することができるようになった。
  • 規定の改良：コンポーネントが改良され、コントローラークラスの再利用が容易になった。
  • セキュリティの強化：XSS（クロスサイトスクリプティング）やCSRF（クロスサイトリクエストフォージェリ）攻撃などの脆弱性を防ぐための機能が追加された。

AWS

AWS（Amazon Web Services）は、Amazonが提供する世界で最も包括的で広く採用されているクラウドサービス

    AWSがクラウドサービスで提供するサービスには、大きく分けて以下のような種類がある。

  • コンピューティング：サーバーやサーバー関連処理を行う
  • コンテナ：コンテナサービスに関する処理
  • ストレージ：ファイルストレージやバックアップに関するサービス
  • データベース：各種データベースサービス
  • 各種移行：環境の移行支援
  • ネットワーク
  • 開発者用サービス
  • 管理とガバナンス：システムコンポーネントの監視や管理など
  • 他、ロボット工学に関するものやブロックチェーン、AI関連の開発などさまざまなサービス

    • このようにAWSでは、コンピューティングやデータベース、ストレージといったクラウドサーバー関連のサービスだけでなく開発環境として使えるサービスや環境移行の支援サービスなど、さまざまなものが提供されている。
